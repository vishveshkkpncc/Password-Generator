# Password Generator

A simple and intuitive web application for generating secure passwords with customizable parameters. Perfect for creating strong passwords for your accounts.

## Features
- 🎚️ **Adjustable Length**: Generate passwords from 1 to 20 characters
- 🔤 **Character Options**: Choose to include or exclude:
  - Uppercase letters (A-Z)
  - Lowercase letters (a-z)
  - Numbers (0-9)
  - Symbols (!@#$%^&*...)
- 💪 **Strength Indicator**: Visual feedback on password strength
- 📋 **One-Click Copy**: Easily copy the generated password to clipboard
- 🎨 **Modern UI**: Clean and responsive design with dark theme

## How to Use
1. Adjust the password length using the slider
2. Select which character types to include
3. Click "Generate" to create a password
4. Click the copy button to copy the password to your clipboard

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript

## Installation
Simply download the files and open `index.html` in your web browser. No installation required!
